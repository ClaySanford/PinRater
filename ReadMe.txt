Discord bot designed to rank pinned messages in a server.
Your bot token needs to be stored in an otherwise empty file title "API.env"
Your guild ID needs to be stored in an otherwise empty file title "guild.env"
Your channel IDs needs to be stored in an otherwise empty file title "channels.env"
Your admin username should be stored in an otherwise empty file titled "user.env" to use the actual ranking commands
!get pulls all pinned messages from all channels with "general" in the name and stores them in a PinList.txt file
!print prints all messages from this txt IFF supplied with a guild and channels enviroment file
